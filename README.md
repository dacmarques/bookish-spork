# Excel Value Counter

This repository contains a simple web-based tool for parsing Excel files and counting occurrences of specific strings in Column A.

## Apps

### Excel Value Counter

Located in `apps/excel-value-counter`, this is a single HTML file application that uses:
- Tailwind CSS (via CDN)
- Phosphor Icons (via CDN)
- XLSX Library (via CDN)

To use the tool, simply open `apps/excel-value-counter/index.html` in a web browser.
